## push()

Push method is used to add things in an array

Example

const numbers = [1,2,3,4];
numbers.push(5);
numbers.push(6,7);

## UPPERCASE

used to capitalize strings in array

Example

const names = ["Douglas","Diana","Donald","Daniel"]

names.toUppercase();

## pop()
Used to remove elements in an array

names.pop()
